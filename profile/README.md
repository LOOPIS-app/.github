# 🌈 LOOPIS introduction

LOOPIS is a non-profit organization established in 2023 in Bagarmossen, Stockholm, Sweden. We make circulation of stuff in a neighbourhood as easy, fun and cheap as possible. Our highly automated web application for ads in combination with a centrally placed locker for exchange creates a hassle-free experience for our members. Our goal is to contribute to a happier and more sustainable future.

## 💾 Environment
We use WordPress as a starting point for our web application, customizing functionality by developing our own modular components (plugins + theme).

## 🧩 Contribution
Introduce yourself in our [Discord server](https://discord.com/channels/1408724249308430378/1408724250323587116) to discuss what you could accomplish. You can also check the list of [issues & feature requests](https://github.com/search?q=org%3ALOOPIS-app+is%3Aissue+is%3Aopen&type=issues). To suggest code changes; create a new branch with your improvements for the component concerned, and make a pull request to 'staging' for review.
